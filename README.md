# Focus Browser

Focus web browser is designed to get rid of complex functionalities of a modern browser. Instead, it uses shortcode combination to navigate web pages.

## Features

### Location (URL) bar

Press Ctrl(⌘)+L to input URL or search text.

### Bookmark

Press Ctrl(⌘)+` then B.

You can register the current page or new one by pressing Ctrl(⌘)+D.

You can assign your own shortcode for the specific bookmark. i.g. If you set duckduckgo.com as D, press Ctrl(⌘)+`, B, then D.

Press Ctrl(⌘)+F to search from your bookmarks.

You can navigate the list with your arrow up/down keys. You can edit(Ctrl(⌘)+Enter) or visit(Space or Enter) it.

### Anchor

Anchor is an instant bookmark that will be removed automatically once you visited the destination. Since Focus is a single tab browser, you can add an anchor in order to revisit the current page.

Press Ctrl(⌘)+/ to register an anchor.

Press Ctrl(⌘)+` then A to see your anchors.

Press Ctrl(⌘)+F or type any letter to activate a find feature.
