## ⚙️ Features

- `Ctrl(⌘)L` Input URL or search text
- `` Ctrl(⌘)` `` Control Centre : You can find more shortcuts from Control Centre to navigate feature
- `Ctrl(⌘)D` Add Bookmark
- `Ctrl(⌘)/` Add Anchor : Instant bookmark that will be removed automatically once you visited the destination
